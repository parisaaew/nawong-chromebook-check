# 🖥️ ระบบตรวจเช็ค Chromebook โรงเรียนวัดนาวง (มัธยมศึกษาตอนปลาย)

ระบบเว็บแอปพลิเคชันสำหรับตรวจเช็คและตรวจสอบสภาพเครื่อง Chromebook และอุปกรณ์ฮาร์ดแวร์ประจำสัปดาห์/ประจำเดือน สำหรับโรงเรียนวัดนาวง ระดับชั้นมัธยมศึกษาตอนปลาย (ม.4 - ม.6)

---

## 🌟 ฟีเจอร์หลักของระบบ (Features)

- 💻 **ตรวจเช็คอุปกรณ์ 4 รายการ:**
  1. ตัวเครื่อง Chromebook (หน้าจอ, คีย์บอร์ด, ทร็คแพด, บอดี้)
  2. สายชาร์จ & อะแดปเตอร์ (Adapter & Cable)
  3. ปากกา Stylus Pen (Stylus Touch Pen)
  4. สติกเกอร์รหัสเครื่อง / ครุภัณฑ์
- 📊 **แดชบอร์ดสรุปภาพรวม (Dashboard Metrics):** สรุปสถานะการตรวจแยกตามห้องเรียน, ประเภทปัญหาฮาร์ดแวร์ และรายการแจ้งส่งซ่อมด่วน
- 📥 **นำเข้าข้อมูลนักเรียนแบบกลุ่ม (Bulk Import):** นำเข้ารายชื่อนักเรียนหลายคนจากไฟล์ Excel / CSV หรือ Copy-Paste ตารางได้ใน 1 วินาที
- 📜 **ประวัติการตรวจเช็คย้อนหลัง (Audit History Logs):** ดูประวัติรอบการตรวจย้อนหลัง สลับไปดูงวดอดีต และพิมพ์รายงานย้อนหลังได้ตลอดเวลา
- 🖨️ **ออกรายงานพิมพ์เอกสาร A4 (Official Printable Report):** พิมพ์แบบบันทึกรายงานฉบับทางการ พร้อมระบบปรับแก้ชื่อและตำแหน่งผู้ลงนามท้ายเอกสาร 3 ฝ่าย (ครูที่ปรึกษา, งาน ICT, ผู้อำนวยการ)
- 🔐 **ระบบเข้าสู่ระบบรักษาความปลอดภัย (Login System):** กำหนดสิทธิ์การเข้าถึงข้อมูลระบบ

---

## 📂 โครงสร้างไฟล์ในโครงการ (Project Files)

- `index.html` — หน้าโครงสร้างหลักของเว็บแอปพลิเคชัน Single Page Application (SPA)
- `style.css` — สไตล์และดีไซน์ระบบ Emerald Slate Theme พร้อมกฎสั่งพิมพ์ A4 (`@media print`)
- `app.js` — ระบบประมวลผล LocalStorage, การบันทึกตรวจเช็ค, คำนวณสถิติ และการจัดการผู้ลงนาม
